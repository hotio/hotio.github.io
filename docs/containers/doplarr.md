## Starting the container

!!! docker ""

    === "cli"

        ```shell
        docker run --rm \
            --name doplarr \
            -e PUID=1000 \
            -e PGID=1000 \
            -e UMASK=002 \
            -e TZ="Etc/UTC" \
            -v /<host_folder_config>:/config \
            ghcr.io/hotio/doplarr
        ```

    === "compose"

        ```yaml
        version: "3.7"

        services:
          doplarr:
            container_name: doplarr
            image: ghcr.io/hotio/doplarr
            environment:
              - PUID=1000
              - PGID=1000
              - UMASK=002
              - TZ=Etc/UTC
            volumes:
              - /<host_folder_config>:/config
        ```
